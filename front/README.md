# Front

**Interface graphique de l'interface admin.**
 

## Preréquis

Avant de lancer il faut au préalable avoir lancé le backend et la base de données mongoDB.

Ensuite, installer toutes les dependences :

```
npm install
```

## Comment utiliser

Lancer l'application avec : 
```
npm start 
```
